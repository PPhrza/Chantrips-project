เว็บไซต์ Chantrips คือเว็บไซต์อะไร?

      เว็บไซต์ Chantrips คือเว็บไซต์แนะนำสถานที่ท่องเที่ยวในจังหวัดจันทุบรี โดยมีการจัดหมวดหมู่เป็นหมวดหมู่ใหญ่ เพื่อให้ตรงกับความสนใจของผู้เข้าเยี่ยมชม สร้างขึ้นด้วยโปรแกรม Visual Studio Code ประกอบด้วยภาษ HTML และ CSS

วัตถุประสงค์ของเว็บไซต์ Chantrips คืออะไร?

      เว็บไซต์ Chantrips สร้างขึ้นเพื่อเป็นเว็บไซต์สำหรับแนะนำสถานที่ท่องเที่ยวในจังหวัดจันทบุรี ทำให้ผู้คนได้รู้จักและเข้ามาท่องเที่ยวในจังหวัดจันทบุรีมากขึ้น และเป็นการกระตุ้นเศรษฐกิจในจังหวัดจันทบุรีอีกด้วย
